# Results Folder

Store publication-ready tables and generated artifacts here.

Recommended layout:

```text
results/
├── paper_tables/                 # static manuscript/reference tables (versioned)
├── training_repro/               # generated by training notebook + checker
│   ├── exp3a_table_test_models_core.csv
│   ├── exp3a_table_test_per_pathway.csv
│   ├── conclusion_check.json
│   ├── comparison_paper_vs_repro_core.csv
│   └── comparison_hybrid_gain_paper_vs_repro.csv
└── generated/                    # optional plotting outputs
```

`training_repro/` is the canonical location for automated conclusion-consistency checks.
`paper_tables/` should store fixed reference values used for release QA comparisons.
